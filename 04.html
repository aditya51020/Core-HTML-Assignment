Ordered Lists

  • <html>: Defines the root of an HTML document.
  • <p>: Represents a paragraph of text.
  • <a>: Defines hyperlinks to other web pages or resources.
  • <img>: Embeds images in the document.
  • <ul>: Creates an unordered (bulleted) list.
  • <ol>: Creates an ordered (numbered) list.
  • <li>: Represents a list item within either an ordered or unordered list.
  • <div>: Defines a division or section within an HTML document, often used for styling and layout purposes.
  • <span>: Inline container used for applying styles or scripting to a specific portion of text.
  • <form>: Contains form elements for user input.
  • <input>: Creates various types of input fields within a form, such as text boxes and checkboxes.
  • <button>: Represents a clickable button.
  • <textarea>: Defines a multi-line text input field within a form.
